Analysis

In analyzing this data, I investigated the breakdown of each of these factors across the US, mapping them among congressional districts and charting their relationships to one another.

Partisan Lean

This first map shows how partisan lean has changed from the years 2018-2022. Using the slider and legend on the right, you can cycle through each of the years and see how the partisan lean values change. Those colored in red hues are those that lean conservative, while those in the blue hues are those that lean democratic. Additionally, you can hover over any district and see the name of the house district, the partisan lean for that year, the change from the previous year (excluding 2018), and its urban-rural class. Additionally, each tooltip contains a line chart that shows how partisan lean has changed over the four years. The colors of the line similarly match the legend and can better help visualize which district remained within the same partisanship and which moved. As we can see from the map, the years of 2018, 2020, and 2021 remained pretty consistent, while many districts swung to the right in 2022.

Change Over Time

The map above explicitly shows these changes between years. The values represent the difference between the previous year and current, while the hues similarly correspond with partisan identity (red representing Republican leaning and blue representing Democratic). Here we are able to see the major change from 2021 to 2022 even more clearly, with only slight changes the years before. Additionally, this chart also contains the option to filter based on urban-rural class to allow a deeper investigation into what districts are changing the most. In using these filters, it appears that the largest changes in partisan leaning are coming from the rural districts, which are mostly becoming more conservative, though not as much as one might expect, as well as the urban districts, which are primarily becoming more liberal.
